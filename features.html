Built for security. Designed for developers.

Every feature works offline, requires zero cloud dependencies, and integrates with any LLM provider.

Core Engine

Multi-layer DLP Engine

📁

2-Layer File Access Gate

Layer 1: filename/extension blocking (.env, .pem, credentials, SSH keys). Layer 2: content DLP scan for PII and credentials. Both always active — no off switch.

⌨️

Exec Command Guard

20+ regex patterns anchored to command position. High-confidence rules block immediately; low-confidence rules get LLM verification via local Ollama. Allows localhost curl, ssh-add, nc -z.

🧠

4-Layer Prompt Analysis

Layer 0: PII regex (<50ms). Layer 1: Intent keywords (weighted scoring). Layer 2: Pydantic rules with negation awareness. Layer 3: Local Ollama LLM for ambiguous cases.

📤

Output Scanner

Scans command output and LLM responses for PII and credentials. Redacts in-place with [REDACTED:PatternName]. Modifying pipeline — doesn't block, fixes.

Privacy Controls

Outbound data protection

🖥️

System Prompt Masking

Strips hostname, OS version, username, repo paths, and shell paths from system messages before they reach the LLM.

📸

Media Filtering

Blocks image, audio, and video attachments from multi-part messages. Prevents visual PII leakage via screenshots or screen recordings.

🔑

Session Encryption

AES-256-GCM encryption for audit log entries and session data at rest. PBKDF2 key derivation from gateway key or auto-generated machine key.

🔐

PII Redaction

Detects and redacts email, phone, SSN, credit cards, API keys, IPv4 addresses in both inputs and outputs. Zero-false-positive philosophy.

🌐

Egress Allowlist

EA_ALLOWED_DOMAINS blocks curl/wget to unlisted external hosts. Localhost always allowed. Prevents data exfiltration to unauthorized domains.

🔗

Taint Tracking

Tags sensitive data at ingress, detects at egress via SHA-256 hash + n-gram Jaccard + substring matching. Cross-session persistence with 24h TTL.

Compliance

Audit-ready from day one

📊

Breach Compliance Engine

13 fintech breach types covering PCI-DSS, SOX, and HIPAA. Rule-based classification — sub-1ms, deterministic, no hallucination risk.

📝

Immutable Audit Trail

SHA-256 checksummed breach records, append-only SQLite, periodic tamper detection scans, and JSONL archival. Every event logged.

🤖

Auto-Instrumentation

Zero-code monkey-patching for OpenAI, Anthropic, and LangChain SDKs. Every LLM call monitored with zero application code changes.

⚠️

Lethal Trifecta Detector

Tracks 3 conditions: private data access + untrusted input + external comm capability. When all 3 met — MCP tools with network are blocked.

Extensions

IDE and browser protection

🧩

VS Code / Cursor Extension

Auto-disables Copilot for sensitive files. Maintains .copilotignore/.cursorignore/.aiignore. Inline secret diagnostics. Write protection — reverts AI edits to sensitive files. @security chat participant for prompt scanning.

🌐

Chrome Extension

DLP guard for browser-based AI tools (Claude web, ChatGPT, Gemini). Prevents PII and credentials from being pasted into LLM web interfaces.
