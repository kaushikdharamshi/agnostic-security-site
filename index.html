The Problem

AI Agents Have Unrestricted Access

Copilot, Claude Code, and Cursor run on developer machines with full access to .env files, API keys, SSH keys, customer PII, and source code. No guardrails.

🔓

Credential Exposure

AI assistants can read .env, .pem, id_rsa, and credentials files — then include them in completions or send them to cloud APIs.

📤

Data Exfiltration

Shell commands like curl -d @secrets.json, printenv, or encoded pipelines can leak data before anyone notices.

⚖️

Shadow AI & Compliance

Unauthorized AI tools running without security team visibility. PCI-DSS, SOX, HIPAA all require controls over sensitive data access.

Architecture

Features

What AgnosticSecurity Protects

📁

File Access Gate

2-layer path + content DLP. Blocks .env, .pem, credentials, SSH keys, and any file containing PII — before the AI sees it.

⌨️

Exec Command Guard

20+ regex patterns with LLM verification. Blocks exfiltration, obfuscated commands, env dumps, and encoded pipelines.

🧠

Prompt Intent Analysis

4-layer detection with 10 decoding sublayers (base64, hex, ROT13, unicode, zero-width chars). Catches evasion attacks.

🔍

Vulnerability Scanner

OWASP Top 10 SAST-lite — catches SQL injection, XSS, command injection in AI-generated code before it hits disk.

🔀

Smart Router

Auto-selects the best model by task type, cost, and latency. 14 models, 5 strategies, automatic failover across providers.

🔒

Privacy Mode

3 enforced modes: full_privacy (local only), balanced (cloud + DLP), permissive (log only). One-command cloud killswitch.

👁️

Shadow AI Detector

Discovers 12+ AI tools running on developer machines (Cursor, Tabnine, Codeium, etc.) and flags unauthorized ones.

🧬

Code Fingerprint Guard

Prevents proprietary code from leaking to cloud LLMs. N-gram Jaccard similarity catches renamed variables.

🕸️

Knowledge Graph

SQLite-backed graph linking agents, threats, data assets, and incidents. Enables "agent X triggered threat Y" correlation.

📝

Security Memory

Obsidian-compatible vault for daily logs, threat intel, incidents, agent profiles, and policy decisions. Human-readable audit trail.

📊

Breach Compliance

13 fintech breach types (PCI-DSS, SOX, HIPAA). Immutable SHA-256 audit log. Configurable taxonomy. Webhook alerting.

✅

Confidence Scoring

Luhn checksum for credit cards, SSA rules for SSNs, Shannon entropy for API keys. Eliminates false positives across all surfaces.
