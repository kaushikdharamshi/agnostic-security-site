FAQ

Common questions

Is the open source version production-ready?

Yes. The Community tier includes the full DLP engine, VS Code extension, and Claude Code hooks. It's the same code that runs in Pro and Enterprise — no artificial limitations.

Does it require internet access?

No. All scanning runs locally. The optional Ollama LLM layer is also local. The only network calls are to your chosen LLM provider — and those go through the security gateway.

Can I use it with Cursor / Windsurf / Cody?

Yes. The VS Code extension supports Copilot, Cursor, Windsurf, and Cody via .copilotignore, .cursorignore, and .aiignore. The DLP engine works with any tool that can call shell commands or HTTP APIs.
